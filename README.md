# 脱出ゲーム

1. 目的・背景
　GitHubを用いたweb脱出ゲームを製作する。ゲームの共通処理を担う「エンジン層」とステージごとの謎の中身を担う「コンテンツ層」に分離する。技術的な学びとステージの量産を分けることで、それぞれの意図を明確化している。また、担当ファイルが物理的に分かれているため、コンフリクトが起きにくいという狙いもある。

2. ゲーム全体仕様
- 4ステージ構成で、プレイヤーは1ステージずつ攻略していく
- 画面には1ステージのみを表示し、ステージをクリアすると「ステージクリア」の演出とともに、次のステージへ進む。
- ステージが進むごとに謎の難易度を上げる
- 最終ステージをクリアすると、全体クリア画面を表示する。
- 一度クリアしたステージには戻れない。
