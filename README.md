## 脱出ゲーム 取扱説明書

1. ゲーム概要
放課後、忘れ物を取りに学校へ戻った主人公は、突然校舎に閉じ込められてしまう。
校内には脱出に必要な鍵が隠されており、プレイヤーは校内を調べながら謎を解き、非常口からの脱出を目指す。
本ゲームはクリック操作のみで進行する探索型脱出ゲームである。

2. ストーリー
放課後、忘れ物を取りに学校へ戻った主人公。
校舎へ入った直後、校内の扉がすべて施錠されてしまう。
校内放送から聞こえてきたメッセージは次の通りである。

「校内に隠された鍵を集めれば出口は開く。」

主人公は図書館と理科室を探索し、謎を解きながら脱出に必要な鍵を集める。

3. 操作方法
マウスクリック

タイトル画面からゲーム開始
オブジェクトを調べる
アイテムを取得する
部屋へ移動する
メッセージを進める

インベントリ
取得したアイテムはインベントリに保存される。
アイテムをクリックすることで説明を確認できる。


4. ステージ紹介
図書館
図書館内に隠されたヒントを探し、本棚に関する謎を解くステージ。
正しい手順で探索を進めることで、非常口の鍵②を入手できる。

理科室
理科室内のメモや薬品棚を調べ、色の順番に関する謎を解くステージ。
正解すると隠された鍵が出現し、非常口の鍵①を入手できる。

非常口
脱出地点。
理科室と図書館で入手した2本の鍵を所持している場合のみ脱出成功となる。


5. クリア条件
以下の条件を満たすこと。

理科室をクリアする
図書館をクリアする
非常口の鍵①を入手する
非常口の鍵②を入手する
非常口を調べる

条件を満たすと脱出成功画面が表示される。
